Quick Start Guide

License Management Automation

Automate Polarion license operations

How to use:
  1. 1. Select an action (Query, Add, Remove, or Bulk)
  2. 2. Paste your license configuration data
  3. 3. Upload user data file if needed (CSV/Excel format)
  4. 4. For user operations, enter email addresses
  5. 5. Click Execute to run the automation
Pro tips:
  • • Use Query action to check current license status
  • • Bulk operations are ideal for large user lists
  • • Always backup license configuration before changes

ReqIF Comparison Tool

Compare ReqIF files and identify differences

How to use:
  1. 1. Upload the original ReqIF file (.reqifz format)
  2. 2. Upload the comparison ReqIF file (.reqifz format)
  3. 3. Click Compare Files to start analysis
  4. 4. Review the detailed comparison report
  5. 5. Export results for documentation
Pro tips:
  • • Ensure both files are in .reqifz format
  • • Larger files may take longer to process
  • • Review high-severity differences first

Frequently Asked Questions

What file formats are supported?

License Management: CSV, Excel files for user data.
ReqIF Comparison: .reqifz files only.

How long do operations take?

Most operations complete within 30 seconds. Large ReqIF files may take up to 2-3 minutes.

Are my files stored securely?

All files are processed locally and temporarily. No data is permanently stored on servers.

Can I export results?

Yes, both tools support exporting results in various formats (CSV, PDF reports).

What if I encounter an error?

Check the error message in the output panel. For persistent issues, contact technical support with the error details.

Can I run multiple operations simultaneously?

Currently, only one operation can run at a time per tool. This ensures data integrity and optimal performance.
